# Dungeon Crawler

Ein wunderbares Projekt, um Programmieren zu lernen

## Getting started

- Repository klonen
- .pro Datei im QtCreator öffnen
- auf "Run" klicken

Gibt auch tausend andere Möglichkeiten, aber so machen wir das

## Steuerung

- 7: nach oben links
- 8: nach oben
- 9: nach oben rechts
- 6: nach rechts
- 3: nach unten rechts
- 2: nach unten
- 1: nach unten links
- 4: nach links
- 5: stehen bleiben
- 0: beenden

## Felder
- . normales Feld
- \# Wand
- O Portal
- _ Grube
- < Rampe
- ? Schalter
- / geöffnete Tür
- X geschlossene Tür
- @ Charakter
- e LootChest
